# Degen Hounds (betting_game-1)

A Next.js site using Chakra UI and Three.js — a small betting/game demo.

## Features

- Next.js + React frontend
- Chakra UI for styling
- Example pages and components ready to modify

## Requirements

- Node.js 14+ and npm

## Install

```bash
npm install
```

## Development

Run the local dev server:

```bash
npm run dev
```

This uses the project's `dev` script which runs `next dev -H 0.0.0.0`.

## Build & Start

Build the production bundle:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## Helpful scripts

- `npm run dev` — start dev server
- `npm run build` — create production build
- `npm run start` — start production server
- `npm run prettier` — format files with Prettier
- `npm run lint` — run ESLint
